# بوابة العملاء V3 - Spare Parts Customer Portal

هذه نسخة مطورة من الداشبورد، مخصصة لتحويل مخزون قطع الغيار إلى بوابة يستطيع العميل البحث فيها وطلب عرض سعر بدون رؤية التكلفة أو قيمة المخزون.

## طريقة التشغيل على جهازك

1. فك الضغط عن الملف.
2. شغل ملف:
   `Install Requirements.bat`
   مرة واحدة فقط.
3. بعد التثبيت شغل:
   `Start Customer Portal.bat`
4. سيفتح الرابط في المتصفح.

## الصفحات الموجودة

### Customer Portal
صفحة آمنة للعميل تعرض فقط:
- Part Number
- Description
- Manufacturer
- Availability

ولا تعرض:
- Cost
- Total Value
- Qty الحقيقي

### Request Quotation
نموذج طلب عرض سعر، ويحفظ الطلبات في ملف:
`data/customer_rfqs.xlsx`

### Admin Dashboard
صفحة داخلية لك تعرض كل البيانات بما فيها التكلفة والقيمة.
كلمة مرور الأدمن الافتراضية:
`1234`

غيرها من ملف `app.py` عند السطر:
`ADMIN_PASSWORD = "1234"`

### Upload Inventory
لرفع ملف مخزون جديد بدون تعديل الكود.

### RFQ Inbox
لعرض وتحميل طلبات العملاء.

## كيف ترفعه أونلاين لاحقًا؟

النسخة الحالية جاهزة محليًا. للنشر كرابط للعملاء، أفضل الخيارات:

1. Streamlit Community Cloud للتجربة.
2. VPS خاص للاستخدام التجاري الجدي.
3. دومين خاص مثل: stock.yourdomain.com

## تنبيه مهم

قبل مشاركة الرابط مع العملاء، لا تنشر نسخة فيها تكلفة أو قيمة المخزون للعامة. هذه النسخة تخفيها من Customer Portal، لكنها تظهر لك فقط في Admin بعد كلمة المرور.
